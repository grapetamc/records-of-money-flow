# records-of-money-flow

Google Apps Script (GAS) + Google Sheets (スプレッドシート) を用いて家計簿的なものを作成した．

GASで生成したWebUIを通じてSheetsに入力できる．

対応している具体的な操作

・入金

　・日付，金額，口座，etcを指定して入金

・出金

  ・日付，金額，口座，etcを指定して出金

・移動

  ・日付，金額，移動元と移動先の口座，etcを指定してお金の移動

入力したデータを煮たり焼いたりすると好みの家計簿ができるでしょう．
